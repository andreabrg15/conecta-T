## Backend (Api)

REST Api desarrollada en Node.js con Express y Prisma ORM, partiendo de las rutas base:

api/usuarios <br>
api/publicaciones <br>
api/comentarios <br>

Ejemplo: <br>


Además manejando el guardado de archivos estáticos.
